# Human Motion Recognition Using IMUs

## Project Description
This project focuses on classifying human activities (walking, sitting, standing, etc.) using Inertial Measurement Unit (IMU) data. We utilize Deep Learning techniques to process time-series sensor data and achieve high-accuracy activity recognition.

## Dependencies
- MATLAB (R2024a or later recommended)
- Statistics and Machine Learning Toolbox
- Deep Learning Toolbox

## Setup Instructions
1. Clone this repository to your local machine.
2. Download the UCI HAR dataset and place it in the `/data` folder.
3. Open MATLAB and navigate to the `/src` folder.

## How to Run
1. Open `main.mlx` in MATLAB.
2. Click "Run" to execute the full processing pipeline.
3. View the generated plots and confusion matrix in the output section.
